## Getting started
### Installation
- Clone this repo:
```bash
git clone <insert-url>
```
- Create a python environment :
```bash
conda env create -f environment.yml
```
Or
```bash
conda create --file requirements.txt
```
- Run a given solution:
```bash
python main.py -n <solution_number>
```
